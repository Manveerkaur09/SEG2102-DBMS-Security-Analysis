# Sevurity and Integrity in Modern DMBMS 

## Overview of the Project
The final assessment submission for the SEG2102 course is supported by this repository.
The project focus on a qualitative and comparative analysis of modern databases security and integrity mechanisme, including Access control, encryption, blockchain-based integrity, and AI-based threat detection. 

## Availability of Code
There is no executable application code or comprehensive system implementation involved in this investigation. As required by the course, the repository is made available for transparency and verification purposes.
